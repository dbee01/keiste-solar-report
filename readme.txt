A free solar calculator and lead generation plugin for solar companies. Calculate potential savings and capture qualified leads.

== Description ==

Keiste Solar Report is a powerful, easy-to-use solar calculator and lead generation plugin designed for solar installation companies and renewable energy businesses.

= Features =

* **Solar Savings Calculator** - Help visitors estimate their potential solar savings
* **Lead Capture Forms** - Collect qualified leads with built-in validation
* **Admin Dashboard** - Manage and export leads easily
* **CSV Export** - Export lead data for CRM integration
* **Email Notifications** - Automatic notifications for new leads
* **Responsive Design** - Mobile-friendly interface
* **No API Keys Required** - Works out of the box with average calculations
* **GDPR Compliant** - Built with privacy in mind

= Shortcodes =

* `[keiste_solar_calculator]` - Display the solar savings calculator
* `[keiste_solar_lead_form]` - Display the lead capture form

= Premium Version =

Upgrade to **Keiste Solar Premium** for advanced features:
* Google Maps Integration
* Google Solar API with real roof data
* Interactive address selection
* Precise solar potential calculations
* Advanced analytics

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/keiste-solar-report/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcodes `[keiste_solar_calculator]` and `[keiste_solar_lead_form]` on any page
4. Configure settings under 'Solar Leads' > 'Settings'

== Frequently Asked Questions ==

= Do I need a Google API key? =

No! The free version uses average calculations and doesn't require any API keys, making it perfect for WordPress.org distribution.

= How do I capture leads? =

Simply add the `[keiste_solar_lead_form]` shortcode to any page. Leads are automatically saved to your dashboard.

= Can I export leads? =

Yes! Go to Solar Leads > All Leads and click "Export to CSV" to download your lead data.

= Is there a premium version? =

Yes! Keiste Solar Premium adds Google Maps integration and real Solar API data for more accurate calculations.

= How do I upgrade to premium? =

Visit Solar Leads > Upgrade in your WordPress admin to learn about premium features and upgrade options.
